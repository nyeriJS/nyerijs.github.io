What We Do

HTML5

We've come a long way since HTML could barely handle a simple page layout. We'll use it to tell websites where you are physically located; to handle high definition video; and to deliver extraordinary graphics

CSS3

Simple to advanced CSS features. Animations | Pagination | Multiple Columns | User Interface Object Fit | FlexBox | Medai Queries | 2D Transforms | 3D Transforms | Web Fonts

JavaScript

Our Main Agenda. We learn and implement JavaScript features for front end development. Use JavaScript to program the behaviour of web Pages. Have a look at JSON | Form Validation using JS | JavaScript Performance.

Bootstrap

Learn how to use Bootstrap to kickstart your websites and applications, including mobile-friendly ones. We will reflect a developer's point of view rather than a designer's.

Node.js

Airbnb, LinkedIn, Walmart, Uber, and even NASA (!) have used Node.js in production. ... We will exercise the lightweight, efficiency of node.js for creating backend systems.

jQuery

We will use the Write Less Do More JavaScript Library to make our code beautiful.

We Think Different That Others Cant

We are commuity that is focused on helping people learn to code and gain skills in Web Develpment and related technologies - HTML5, CSS3, JavaScript and its related technologies.

Join us afternoons weekdays from 2PM at the Resource Center for free workshops, info sessions and tech talks. Our events are well-suited for both beginners learning to code as well as intermediate devs aiming to sharpen their skills.

Our meetups Topics Include:

  1. HTML 5 Web APIs
  2. HTML/CSS lessons, tips and tricks using Bootstrap for fast app development
  3. NodeJS for back-end systems
